Kernel Cormas

Si vous n'avez jamais installer Cormas sur VisualWorks, faites une installation complète de Cormas.

version actuelle: décembre 2005

Comment charger un nouveau Kernel Cormas ?

  • télécharger le fichier upgradeCormas.zip (253 Ko) et dézipper le dans Disk:\VW7.3\cormas\ (écraser les fichier existants).

  • ouvrir cormas.im,

  • fermer toutes les fenêtres (... sauf la principale : VisualLauncher, bien sûr),

  • charger le nouveau noyau de Cormas :
    • Menu -> Tools -> Cormas -> install Cormas
    • choisir le fichier Kernel-v.vendredi 2 décembre 2005 - 55140.st du répertoire Disk:\VW7.3\cormas\kernel\
      Cette opération est assez longue...


C'est fini. N'oublier pas de sauver votre image "Cormas2004.im"
File --> SaveAs... du menu de VisualLauncher.
