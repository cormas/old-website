Téléchargement les premiers exemples de modèles

Pour essayer Cormas, vous pouvez télécharger les modèles suivants :

  • trois modèles didactiques
    • Un exemple d'automate cellulaire [FireAutomata],
    • Un exemple d'agents situés [Pursuit],
    • Un exemple d'agents communicants [PlotsRental]
    • Un exemple d'entités spatiales agrégées [TSE]
  • deux modèles théoriques
    • Le fameux modèle d'Axtell et Epstein [SugarScape]
    • Emergence de conventions sur les ressources communes [Dricol]
  • un modèle appliqué
    • Chasse de céphalophes bleus dans les forêts du Cameroun [Djemiong]

Démarche à suivre pour installer des modèles: cliquer ici.

D'autre part, une bibliothèque de modèles téléchargeables est disponible à partir des pages applications. Certains modèles se présentent sous la forme d'archives zip.

Nous vous invitons à vous inscrire sur le forum Cormas, qui vous permettra de vous informer et de communiquer vos commentaires aux autres utilisateurs.
