Installation complète de VW et de Cormas

1. Installer VisualWorks

Cormas a été développé avec la version 7.6 non-commerciale de VisualWorks® (Cincom Systems), qui doit donc être installée sur votre machine. Comme la version 7.6 n'est plus disponible sur le site de Cincom, et à la condition de vous engager à ne pas utiliser VisualWorks à des fins commerciales, vous pouvez directement télécharger vw7.6nc (version Windows + Mac OS X) depuis le site de Cormas. Il est conseillé de dézipper l'archive directement sous la racine du disque dur principal de votre ordinateur pour minimiser les problèmes de configuration récalcitrante (!) qui peuvent survenir avec certains systèmes d'exploitation.

2. Installer Cormas (VERSION : Cormas v.2023.02)

2.a Installer Cormas pour la première fois
  • Télécharger la dernière version de Cormas (cormas2023_Full.zip, 21 Mo, février 2023) qui contient le répertoire complet de Cormas.
  • Dézipper l'archive dans le répertoire VW7.6nc\. Le sous-répertoire 'cormas' est créé. Il contient le fichier cormas.im (des instructions en anglais sont détaillées ici). Copiez vos modèles dans le sous-répertoire VW7.6nc/cormas/Models/.
  • (Conserver votre ancienne version de Cormas. Si une ancienne version de Cormas est déjà installé sur votre machine, vous pouvez garder cette version. Changez simplement le nom du répertoire 'Cormas' en 'Cormas_Old' ou autre nom).
ou 2.b Mettre à jour Cormas
  • Si Cormas est déjà installé, faire une simple une mise à jour en dézippant l'archive cormas2023.zip (2 Mo contenant seulement la "parcel" Cormas) dans le répertoire VW7.6nc/cormas/.
  • Ouvrir cormas.im et sélectionner le menu '?' et 'Update Cormas' afin de mettre à jour le logiciel.
ou 2.c Installation alternative : Store
  • Lorsque VisualWorks est installé, vous pouvez également télécharger et installer Cormas depuis le "Public Repository" de Store. Depuis la fenètre de VW, connectez-vous sur Store (Menu -> Store -> Connect to Repository...), puis sélectionnez "Cincom Public Repository" et Connect. A nouveau dans le menu principal : Menu -> Store -> Published items, rechercher cormas et chargez-le.
