VW 7.6nc

Télécharger VisualWorks

Cormas a été développé avec la version 7.6 non-commerciale de VisualWorks® (Cincom Systems), qui doit donc être installée sur votre machine.

A la condition de vous engager à ne pas utiliser VisualWorks à des fins commerciales (veuillez visiter la Personal Use License de Cincom), vous pouvez directement télécharger l'environnement de programmation vw7.6nc (nc = non-commerciale) depuis le site de Cormas :

Version pour Windows : vw7.6nc (version Windows)
Version pour Mac OSX : vw7.6nc (version Mac OS X)
Version pour Linux : vw7.6nc (version Linux)

Il est conseillé de dézipper l'archive directement sous la racine du disque dur principal de votre ordinateur pour minimiser les problèmes de configuration récalcitrante qui peuvent survenir avec certains systèmes d'exploitation.

Alternativement, télécharger l'image iso de 7.6.

 

Filmez vos simulations

Si vous souhaitez prendre des photos de la grille ou faire des films, cette option (uniquement disponible sous Windows) nécessite d'avoir installer QuickTime. Il faut également télécharger un fichier DLL (JunWinQT.dll), à sauver dans Disk:\VW7.6\bin\win
