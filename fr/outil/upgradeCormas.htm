Mise à jour de Cormas2004

Si vous n'avez jamais installer Cormas sur VisualWorks, faites une installation complète de Cormas. Autrement, vous devriez avoir dans votre répertoire Disk:\VW7.3\cormas\, le fichier cormas2003.im (ou cormas2004.im), ce qui signifie que votre configuration de VisualWorks doit être réussie. Dans ce cas, vous avez simplement besoin de changer dans votre image, la version de votre noyau de Cormas par celle incluse dans cette page.

Dans le menu principal de Cormas, sélectionnez Help --> aboutCormas. Si la date affichée est antérieure au 4 décembre 2006, vous avez donc besoin de faire une mise à jour de Cormas.

Il y a 2 possibilités :

  • Charger une parcel Cormas (méthode la plus rapide) ou
  • Charger le nouveau noyau (Kernel Cormas : méthode beaucoup plus lente).

Cette nouvelle version permet de faire des films et des photos de la grille de Cormas. Cette option nécessite QuickTime et un fichier DLL (JunWinQT.dll) à sauver dans le répertoire Disk:\VW7.3nc\bin\win.
