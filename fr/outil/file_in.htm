Comment intégrer un "fichier.st" dans Cormas ?

 

Un fichier.st (st = smalltalk) contient le texte des méthodes qui sont à corrigées ou à ajouter. Pour intégrer un fichier.st à Cormas, nous présentons ici les étapes :

  1. Dans le menu de la fenêtre principale de VisualWorks, selectionnez File --> File Browser ou cliquez sur la première icône .



  2. Un navigateur de fichier ("File Browser") s'ouvre alors. Il pointe sur votre "cormas".

    Naviguez jusqu'au répertoire de votre choix et sélectionnez le fichier à charger

  3. Avec le bouton droit de la souris sélectionnez la commande "file in".


  4. Ca y est ! Fermer la fenêtre "File In" et sauvez votre image.
