Importer des modèles dans Cormas

La démarche à suivre pour installer des modèles dans Cormas est simple :

  • Créez un sous-répertoire de Cormas et nommez-le Models. Exemple :

    C:\vwnc30\cormas\Models

  • Dézipper les archives téléchargées (file.zip) dans ce répertoire.
  • Dans le menu de Cormas, cliquez sur Importer.


  • Une fenêtre s'ouvre alors. Double-cliquez sur le modèle qui vous intéresse.


  • Le modèle est chargé ! Par la suite, vous n'aurez plus qu'à le réouvrir.
