Parcel Cormas

Si vous n'avez jamais installer Cormas sur VisualWorks, faites une installation complète de Cormas.

version actuelle: décembre 2005 parcelCormas.zip

Comment charger la parcel Cormas ?

  • télécharger le fichier parcelCormas.zip et dézipper le dans Disk:\VW7.3\cormas\ (écraser les fichiers existants: cormas.pcl, cormas.pst et tout le répertoire Messages).
  • ouvrir l'image de Cormas (cormas2003.im ou cormas2004.im)
  • fermer toutes les fenêtres (... sauf la principale : VisualLauncher, bien sûr),
  • Menu -> Tools -> Cormas -> update Cormas
  • Si le menu "Update Cormas" n'apparaît pas, regarder ici...
  • C'est fini. N'oublier pas de sauver votre image "Cormas2004.im"
