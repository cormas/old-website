Parcel Cormas

Comment charger la parcel Cormas ?

  • télécharger le fichier parcelCormas.zip et dézipper le dans Disk:\VW7.3\cormas\ (écraser les fichiers existants).
  • ouvrir l'image de Cormas (cormas2003.im ou cormas2004.im)
  • fermer toutes les fenêtres (... sauf la principale : VisualLauncher, bien sûr),
  • Si une parcel Cormas est déjà chargée, supprimer la :
    • Menu -> Tools -> Unload Parcel Named ...
  • puis, charger la parcel cormas. 2 possibilités :
    • Menu -> Tools -> Load Parcel Named ...

      et tapper "cormas" dans la fenetre :
      puis
    • Ou bien, clicker sur le bouton "open a Parcel manager"
      Une fenêtre s'ouvre :

      Aller sur l'onglet "Directories" et sélectionner le dernier répertoire ".".
      Sélectionner la parcel cormas et clicker sur le bouton Load.
  • C'est fini. N'oublier pas de sauver votre image "Cormas2004.im"
    File --> SaveAs... du menu de VisualLauncher.
