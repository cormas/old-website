Télécharger Cormas2002 - bêta

Merci par avance à tous les bêta testeurs.

Les nouvelles caractéristiques de Cormas2002 :

Une petite présentation (en français) est disponible : presentationCormas2002.ppt

Cormas2002 tourne sur la dernière version de VisualWorks: VW5i.4. Les nouvelles fonctionnalités de cette version sont donc présentes dans Cormas2002, par exemple les NameSpaces, connexions aux bases de données (relationnelles et objets), les
protocoles de communication sur internet, etc. VisualWorks est compatible avec Windows® (95/98/ME/NT/2000), PowerMac, Intel Linux, AIX, SGI Irix, Compaq UNIX, HP-UX, et Solaris.

Cormas2002 contient d'autres caractéristiques :

  • possibilité de faire tourner un modèle sans afficher la grille, ce qui augmente la vitesse des simulations,
  • possibilité de conduire des analyses de sensibilité plus facilement,
  • des connexions avec Excel, permettant de sauver les données de simulation directement dans des tableaux,
  • des facilités pour construire des agrégats spatiaux, à partir de cellules régulières ou irrégulières,
  • possibilité de sauver et de charger des agrégats avec leur composants et leur voisinage
  • possibilité de travailler avec différentes version d'un modèle,
  • un bouton "pause"
  • versionnement précis de Cormas
  • une procédure simplifiée pour installer Cormas2002 (Parcel CORMAS)

L'importation de vos modèles depuis VW3 vers VW5i.4 est géré par Cormas2002, sans manipuler votre code (normalement !).
D'autres outils seront délivrés progressivement, comme la connexion aux BD, la connexion aux SIG, un "movie make", des indicateurs spatiaux, etc.
Une documentation sera bientôt disponible avec la description de l'architecture de Cormas et des explications sur les principales
méthodes.

Installation de Cormas2002

Si vous n'avez jamais installer Cormas2002 sur VW5i.4, faite une installation complète de Cormas2002.

Si vous avez déjà installer une version de Cormas2002 sur VW5i.4, faite une simple mise à jour de votre version.
