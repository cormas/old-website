Pursuit

Poursuite entre prédateurs et proies

Christophe Le Page, François Bousquet (Cirad)

Ce modèle illustre les principes des agents situés de Cormas. Les agents situés ont un attribut patch qui les relie à l'entité spatiale sur laquelle ils sont situés. L'entité spatiale élémentaire de Cormas possède un attribut theOccupants qui est un registre de tous les agents situés dessus. Ce système de double-référencement permet aux agents situés de prendre connaissance des autres agents qui les entourent.

Dans ce modèle très simple, les agents prédateurs attrappent les agents proies situées sur la même cellule. Il n'y a pas d'autre forme de mortalité, pas de reproduction. Le nombre de prédateurs ne varie donc pas, le nombre de proies diminue plus ou moins rapidement selon les facultés de déplacement de chacun des deux types d'agents. Le modèle propose trois niveaux de perception des prédateurs par les proies et deux niveaux de perception des proies par les prédateurs.
