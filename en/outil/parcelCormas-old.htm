Loading a new Cormas Parcel

  • Download the archive file parcelCormas.zip and unzip it into Disk:\VW7.3\cormas\
  • Open cormas.im
  • Close all windows (... except the main one : VisualLauncher, of course),
  • If a Cormas parcel has already been loaded, you must remove it first :
    • Menu -> Tools -> Unload Parcel Named ...
  • Load the Cormas parcel. 2 possibilities
    • Menu -> Tools -> Load Parcel Named ... from the main VisaulWorks menu porposed in the VisualLauncher window)

      and write "cormas" in the pop-up window :
      then
    • Or, click on "open a Parcel manager" button
      A new window pops up :

      Select "Directories" item and select the last directory called ".".
      Select the cormas parcel and click on Load button.

  • That's all ! Do not forget to save your image Cormas2004.im (File --> SaveAs... from the main VisualWorks menu porposed in the VisualLauncher window).

This new version makes it possible to take snapshots and films of the Cormas grid (this option requires QuickTime and a DLL file (JunWinQT.dll) to be saved in Disk:\VW7.2\bin\win directory).
