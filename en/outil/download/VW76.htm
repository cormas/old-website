Cormas has been developped with the non-commercial version of VisualWorks® from Cincom Systems, which is therefore needed to be installed on your computer.

Version for Mac OSX: vw7.6nc (version Mac OS X)
Version for Linux: vw7.6nc (version Linux)

You can unzip the file directlyon the root of your hard drive.

Alternatively, download the iso CD of 7.6nc.

 Making movies

To snap the grid or make movies (only available on Windows), it is necessary to install QuickTime. Then you need to download the DLL file (JunWinQT.dll) on Disk:\VW7.6\bin\win
