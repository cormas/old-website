Upgrading Cormas Kernel

If you have never installed Cormas in the VisualWorks environment, you should go to the full install page.

Current version: December, 2005

Loading a new Cormas Kernel

  • Download the archive file upgradeCormas.zip (253 Kb) and unzip it into
    Disk:\VW7.3\cormas\ (overwrite the existing files).
  • Open cormas.im

  • Incorporate the new Cormas kernel
    • Tools -> Cormas -> install Cormas from the main VisaulWorks menu porposed in the VisualLauncher window)
    • select the file Kernel-v.vendredi 2 décembre 2005 - 55140.st located in the directory Disk:\VW7.3\cormas\kernel
      (Be careful, this operation is rather long...)


That's all ! Do not forget to save your image Cormas2004.im (File --> SaveAs... from the main VisualWorks menu porposed in the VisualLauncher window).

This new version makes it possible to take snapshots and films of the Cormas grid (this option requires QuickTime and a DLL file (JunWinQT.dll) to be saved in Disk:\VW7.3\bin\win directory).
