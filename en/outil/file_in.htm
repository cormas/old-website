How to integrate a "file.st" into Cormas ?

 

A file.st (st = smalltalk) contains the text of the methods we want to correct or to add. In order to explain the way to add this files to Cormas, we present here the main steps :

  1. In the main menu of the VisualWorks Launcher, select File --> File Browser or click on the first icon .



  2. A new file browser pops up pointing on "cormas" directory.

    Navigate to the directory of your choice then select the file to load.

  3. Select the command "file in" with the rigth button of the mouse.



  4. That's it ! Close the browser and save your image.
