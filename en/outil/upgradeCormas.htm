Upgrading Cormas2004

 

If you have never installed Cormas in the VisualWorks environment, you should go to the full install page. Otherwise, there should exist a cormas2003.im or (cormas2004.im) file located in the directory Disk:\VW7\cormas of your computer. This means that the configuration of VisualWorks should be ok. Then you simply need to replace the kernel of Cormas that is incorporated into this image by the new one proposed here.

In the main Cormas menu, select Help --> aboutCormas. If the date that is displayed is prior to december 4, 2006, then you need to upgrade your Cormas.

2 possibilities to upgrade are available :

  • from the cormas parcel (the quickest way) or
  • from the new kernel (the slowest way)

This new version makes it possible to take snapshots and films of the Cormas grid (this option requires QuickTime and a DLL file (JunWinQT.dll) to be saved in Disk:\VW7.3\bin\win directory).
