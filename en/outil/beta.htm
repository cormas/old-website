The Cormas2002 new features

A short presentation (in french) : presentationCormas2002.ppt.

The Cormas2002 version works with the most recent version of VisualWorks: VW5i.4.
The new advantages of this version are consequently present in Cormas2002, for example the NameSpaces definition, connectivity to all major relational databases, object databases, and internet standard protocols. VisualWorks is compatible with Windows® (95/98/ME/NT/2000), PowerMac, Intel Linux, AIX, SGI Irix, Compaq UNIX, HP-UX, and Solaris.

Cormas2002 provides other new features such as

  • the possibility to run a model without displaying the grid in order to increase the simulation speed,
  • the possibility of conducting sensitivity analysis more easily,
  • Connections with Excel, allowing to save the data into Excel worksheets,
  • Some facilities to build spatial aggregats, from regular or irregular cells,
  • Saving and loading aggregats with their components and their neighboors,
  • the ability to works with different versions of a model,
  • the "pause" button
  • Cormas versionning
  • an easier installation of Cormas2002 (CORMAS Parcel)

The importation of your models from VW3 to VW5i.4 is dealt with by Cormas2002, without munipulation of your code (normally!)

Other features will be delivered progressively such as DB-connectivity, GIS-connectivity, a movie make, landscapes indices, etc. A documentation will be available soon, describing architecture of Cormas and explaining a lot of methods.

Installing Cormas2002

If you never install Cormas on VW5i.4, go on the page of complete installation.

If you have already install a previous version of Cormas2002 on VW5i.4, just upgrade to the current version.
